# Leia-me para utilização.
--

Para utilização correta do software, é necessarío executar o arquivo ip.ps1 via Powershell.

# Procedimento em Windows 10 e/ou Windows Server 2012/2016/2019 com linguagem em PT-BR
Clique com o botão direito sobre o arquivo, e clique em "Executar com o Powershell"

# Procedimento em Windows 10 e/ou Windows Server 2012/2016/2019 com linguagem em EN-US
Clique com o botão direito sobre o arquivo, e clique em "Run with Powershell"

--
#
--

Por padrão, o software vem selecionado a opção "Automático", podendo ser alterado para "Manual".
Para utilizar a função automático, é necessário clicar no botão "Setar IP Interno" para buscar o IP da máquina que está respondendo ao IPv4.
Em seguida, inserir somente o DNS que será ajustado no Host do Windows.

-- Obs.: Caso houver muitas placas de rede ativas na estação que estiver sendo executado o software, poderá interpretar de forma errado o IPv4. --

Para utilizar a função manual é necessário apenas informar manual o IP que será incluído no Host do Windows.
Em seguida, inserir somente o DNS que será ajustado no Host do Windows.
